1 变量和简单的数据类型

1.1变量

1.1.1 变量的命名和使用

  • 变量名只能包含字母，数字和下划线。变量名可以字母和下划线打头，但不能以数字打头。
  • 变量名不能包含空格但是能使用下划线分割单词。
  • 不要讲Python关键字和函数名作为变量名
  • 变量名应简短又具有描述性
  • 慎用小写字母i和大写字母o
  1. 将一条消息存储到变量中在将其打印出来

1.2 字符串

1.2.1修改字符串的大小写

  • 在print()语句中，title()出现在这个变量后面
    • title()以首字母大写的方式显示每个单词
    • upper()所字母以大写方式显示
    • lower()所有字母以小写方式显示

1.2.2合并字符串

#注释1

#注释2

  • 使用（+）号来合并字符串full_name=first_name + fast_name来实现完整显示姓名（注释1）

  • 也可以使用储存在变量中的信息来创建完整的消息（注释2）

1.2.3使用制表符或换行符来添加空白

  • 字符串中添加制表符：\t
  • 字符串中添加换行符：\n

1.2.4删除空白

  • 同时剔除字符串两端的空白可以使用lstrip()和rstrip()

1.2.5使用字符串避免语法错误0

  • 在单引号括起来的字符中如果出现撇号将导致错误第一个单引号和撇号之间的内容视为一个字符，进而将于下的文本视为Python代码从而引发错误

1.3 数字

1.3.1整数

  • Python中，可对整数进行加(+)减(-)乘(*)除(/)
  • 二个**表示乘方 一个表示相乘

1.3.2浮点数

  • 带小数点的数字都称为浮点数

1.3.3使用函数str()避免类型错误

  • 在消息中使用变量时，可调用函数str（）讲整数表示为字符串
  • 用法：str（变量）
